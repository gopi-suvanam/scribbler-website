What Scribbler Does

Scribbler is a web-native, executable JavaScript notebook with:

Live Code Cells

Run JavaScript instantly in isolated cells with shareable state and persistent snippets.

Visual Outputs

Built-in support for charts, DOM previews and custom renderers for libraries like D3 and Three.js.

AI Assistants

Context-aware code suggestions, example generation, and auto-documentation to accelerate learning and prototyping.

Easy Sharing

Share runnable notebooks with a link, embed in docs, or export for teaching labs and bootcamps.
